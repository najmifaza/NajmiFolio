# 🎵 NiceGuy SoundBoard

**NiceGuy SoundBoard** is a simple interactive web application that allows users to instantly play various popular meme sound effects. It is built using React and Vite, featuring modern styling with Tailwind CSS.

This project is designed for speed and ease of use, supporting audio playback via button clicks and **keyboard shortcuts**.

## 🌟 Key Features

- **Instant Audio Playback**: Plays short audio clips (meme sounds) without delay.
- **Keyboard Shortcuts**: Press specific keyboard keys to play sounds without clicking (Keybinds are listed below).
- **Audio Visualization**: Simple animated bar effects on the card while the audio is playing.
- **Responsive Design**: A neat grid layout utilizing Tailwind CSS.
- **Playback Management**: Toggle feature (Play/Pause) and automatic reset when audio is replayed.

## 🛠️ Technologies Used

This project is built using a modern technology stack:

- **[React](https://react.dev/)** (v19) - The core UI library.
- **[Vite](https://vitejs.dev/)** - A fast build tool for frontend development.
- **[Tailwind CSS](https://tailwindcss.com/)** (v4) - A utility-first CSS framework for styling.
- **[Lucide React](https://lucide.dev/)** - A lightweight and flexible icon collection.
- **Framer Motion / CSS Animation** - For visual effects on the sound cards.

## 🚀 How to Run (Installation)

Follow these steps to run the project on your local machine:

### 1. Clone the Repository

```bash
git clone [https://github.com/najmifaza/soundboard.git](https://github.com/najmifaza/soundboard.git)
cd soundboard

```

### 2. Install Dependencies

Ensure you have [Node.js](https://nodejs.org/) installed. Use `npm` or `pnpm` (recommended if there's a `pnpm-lock.yaml`).

```bash
npm install
# or
pnpm install

```

### 3. Run Development Mode

```bash
npm run dev
# or
pnpm dev

```

Open your browser and navigate to the address shown in the terminal (usually `http://localhost:5173`).

### 4. Build for Production

If you want to deploy the application:

```bash
npm run build

```
