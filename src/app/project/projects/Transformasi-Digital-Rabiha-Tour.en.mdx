## Overview

This project aimed to build the credibility of Rabiha Tour & Travel through a solid digital identity. I was fully responsible for everything from designing branding assets and managing visual content on Instagram, to developing a website that serves as an information hub for Umrah and Hajj packages for prospective pilgrims.

## Key Features

- **Visual Branding & Identity**: Designed a consistent brand aesthetic using a combination of CorelDRAW for vector elements and Photoshop for photo processing. The main focus was to create an impression that is religious yet modern and professional.
- **Instagram Content System**: Developed an informative and structured feed design. I created custom templates for departure packages, worship tips, and pilgrimage info to ensure the Instagram layout remains neat and aligned with the brand identity.
- **Informative Web Design**: Built a website using WordPress and Elementor focused on ease of readability. The website is designed to transparently display itinerary details, hotel facilities, and company legality.
- **Direct Interaction Integration**: Integrated a navigation system that makes it easy for prospective pilgrims to directly connect with admins via WhatsApp, considering the characteristics of the target audience who are more comfortable consulting directly.

## Technologies Used

- **WordPress & Elementor**: The main platform to build a responsive website that is easy for the operational team to manage independently.
- **CorelDRAW**: Used for creating digital brochure layouts, logo design, and vector-based graphic elements that require high precision.
- **Adobe Photoshop**: Used for editing departure documentation photos, creating website banners, and visual manipulation for social media promotional content.

## Challenges and Learnings

The main challenge in this project was compiling highly detailed travel package information so that it remains attractive and easy to understand on mobile screens. I learned to optimize the navigation flow in Elementor so that prospective pilgrims can find departure schedules in just two clicks. Because the website is still in the developing stage, I continue to test the design's responsiveness.

## Outcome

Even though the website is still in the development stage before its official release, the branding design and social media content have successfully improved the professional image of Rabiha Tour & Travel. The design consistency I applied from CorelDRAW to the WordPress layout creates a strong and trusted visual identity for prospective customers.
