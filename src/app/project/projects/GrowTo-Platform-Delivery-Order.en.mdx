## Overview

GrowTo is a directory web project designed to make it easier for Purwokerto residents to find delivery order services within a single integrated platform. This project was developed as the Final Project for the Soedirman Digital School (SDS) 2025 in the Website Development path. Over 1.5 months, along with my team member, Zahratul Askia, I participated in intensive mentoring from Special Skill Indonesia to transform this concept into a functional digital product.

- **Directory Listing System**: Implemented a neat directory system to categorize various delivery services, allowing users to search for services based on their specific needs in the Purwokerto area.
- **Responsive Landing Page**: Built a modern and mobile-friendly main interface using the Astra Theme, ensuring excellent accessibility for smartphone users.
- **Custom Layouting**: Used Elementor to design an intuitive visual layout, combining design aesthetics with ease of navigation for everyday users.
- **Information Architecture**: Structured service information so that contact details and delivery service types are presented transparently and are easy to find.

## Technologies Used

- **WordPress**: As the primary Content Management System (CMS) for managing directory data.
- **Elementor**: For a custom and interactive User Interface (UI) design.
- **Astra Theme**: As a lightweight base theme to optimize website performance and loading speed.
- **Directory Plugins**: Used to manage listing features and the delivery services database systematically.

## Challenges and Learnings

The main challenge in this project was completing the product development cycle within a relatively short timeframe (1.5 months) while ensuring the validity of the service data displayed. Through the mentoring process at SDS 2025, I learned a lot about team collaboration in web development, choosing efficient plugins, and aligning functionality with the real needs of the local community in Purwokerto.

## Outcome

Project GrowTo was successfully completed as a proof of my competence in the web development workflow using WordPress. This platform served not only as a graduation requirement for the program but also as a prototype of a relevant digital solution to support the MSME ecosystem and local delivery services in Purwokerto.
