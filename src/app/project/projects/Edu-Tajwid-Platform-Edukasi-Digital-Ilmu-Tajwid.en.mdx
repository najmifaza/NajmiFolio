# TajwidApps (Edu-Tajwid)

**TajwidApps** is a web-based educational platform designed to help users learn the rules of reading the Qur'an (Tajweed), articulation points of letters, and characteristics of letters fluently and precisely.

This application is built as a modern, responsive, and interactive _Static Web Application_, making it easier for users to learn tajweed anywhere via desktop or mobile devices.

---

## Key Features

### 1. Structured & Comprehensive Learning

Provides a systematically arranged curriculum from basic to advanced levels. The material is divided into 16 main chapters covering all essential aspects of Tajweed science.

### 2. Serverless Contact Form (Web3Forms)

Utilizes **Web3Forms** integration to handle messages from users directly to email without requiring a PHP backend, keeping the application lightweight and static.

### 3. Modern & Responsive Interface

- **Adaptive Design**: Built using **Bootstrap v5.3.3**, ensuring the site layout automatically adjusts across various screen sizes (Mobile, Tablet, Desktop).
- **Mobile-Friendly Navigation**: Easily accessible navigation menu with _off-canvas_ support for mobile devices.

### 4. User Interactivity

- **Smooth Animations**: Uses **AOS (Animate On Scroll)** to provide attractive visual effects as users scroll through the page.
- **Media Gallery**: Integration of **GLightbox** to display learning videos or illustrations without leaving the main page.
- **Statistics & Sliders**: Displays statistical data with **PureCounter** and user reviews using **Swiper**.

---

## Course Curriculum

This application covers 16 Tajweed topics available on separate pages:

1.  **Rules of Nun Sukun and Tanwin** (Izhar, Idgham, Iqlab, Ikhfa)
2.  **Rules of Mim Sukun**
3.  **Rules of Mim Tasydid and Nun Tasydid**
4.  **Rules of Lam Tarif** (Syamsiyah & Qamariyah)
5.  **Rules of Thick and Thin Lam** (Lafzul Jalalah)
6.  **Idgham Mutamatsilain**
7.  **Idgham Mutaqaribain**
8.  **Idgham Mutajanisain**
9.  **Rules of Prolongation (Mad)**
10. **Rules of Reciting Ra**
11. **Rules of Qalqalah**
12. **Imalah and Isymam** (Gharib)
13. **Naql and Tashil** (Gharib)
14. **Saktah and Rules for Reading Sad as Sin**
15. **Mad Fathah Read Short**
16. **Rules of Waqaf and Ibtida'**

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla).
- **CSS Framework**: Bootstrap v5.3.3.
- **Form Handling**: [Web3Forms](https://web3forms.com/).
- **External Libraries**:
  - `AOS` - Scrolling animations.
  - `GLightbox` - Media gallery lightbox.
  - `Swiper` - Responsive touch slider.
  - `PureCounter` - Number counting animations.
