## 1. Konfigurasi & Dasar (Setup)

### Directives

Wajib ada di file CSS utama.

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

### Nilai Arbitrary (JIT Mode)

Gunakan kurung siku `[]` untuk nilai kustom yang tidak ada di tema default.

- `w-[350px]` (Lebar pixel spesifik)
- `bg-[#1da1f2]` (Warna Hex kustom)
- `top-[17px]` (Posisi spesifik)
- `grid-cols-[200px_1fr]` (Grid kustom)
- `content-['hello']` (Konten pseudo-element)

## 2. Tata Letak (Layout) & Posisi

| Fitur                  | Class                                                       | Deskripsi                                                             |
| ---------------------- | ----------------------------------------------------------- | --------------------------------------------------------------------- |
| **Container**          | `container`                                                 | Mengatur lebar maksimal sesuai breakpoint (biasanya butuh `mx-auto`). |
| **Display**            | `block`, `inline`, `inline-block`, `flex`, `grid`, `hidden` | Mengatur tipe tampilan elemen.                                        |
| **Position**           | `static`, `fixed`, `absolute`, `relative`, `sticky`         | Mengatur posisi elemen.                                               |
| **Top/Right/Btm/Left** | `top-0`, `right-4`, `bottom-auto`, `inset-0`                | Posisi koordinat elemen (jika absolute/fixed/relative).               |
| **Z-Index**            | `z-0`, `z-10`, `z-50`, `z-auto`                             | Mengatur tumpukan elemen (layering).                                  |
| **Overflow**           | `overflow-auto`, `overflow-hidden`, `overflow-x-scroll`     | Mengatur konten yang meluap.                                          |

## 3. Ukuran (Sizing)

| Tipe               | Class                                                   | Deskripsi                               |
| ------------------ | ------------------------------------------------------- | --------------------------------------- |
| **Width**          | `w-0`, `w-full`, `w-screen`, `w-1/2`, `w-auto`          | Lebar elemen (skala 0-96, %, viewport). |
| **Height**         | `h-0`, `h-full`, `h-screen`, `h-1/2`, `h-auto`          | Tinggi elemen.                          |

| **Max Width**      | `max-w-xs`, `max-w-md`, `max-w-full`, `max-w-screen-xl` | Batas maksimal lebar.                   |
| **Min/Max Height** | `min-h-screen`, `max-h-full`                            | Batas minimal/maksimal tinggi.          |

## 4. Flexbox & Grid

### Flexbox

- **Container**: `flex`, `inline-flex`
- **Direction**: `flex-row`, `flex-row-reverse`, `flex-col`, `flex-col-reverse`
- **Wrap**: `flex-wrap`, `flex-wrap-reverse`, `flex-nowrap`
- **Grow/Shrink**: `grow`, `grow-0`, `shrink`, `shrink-0`
- **Justify (Main Axis)**: `justify-start`, `justify-center`, `justify-between`, `justify-around`, `justify-end`
- **Items (Cross Axis)**: `items-start`, `items-center`, `items-end`, `items-baseline`, `items-stretch`

### Grid

- **Container**: `grid`
- **Columns**: `grid-cols-1`, `grid-cols-4`, `grid-cols-12`, `grid-cols-none`
- **Rows**: `grid-rows-1`, `grid-rows-4`, `grid-rows-none`
- **Span**: `col-span-1`, `col-span-2`, `col-span-full`, `row-span-2`
- **Gap**: `gap-1`, `gap-4`, `gap-x-2`, `gap-y-4`

| ----------------- | ---------------------------------------------- | ------------------------------------------- |

## 7. Desain Responsif (Breakpoints)

Gunakan prefix untuk menerapkan style pada ukuran layar tertentu ke atas (**Mobile-First**).

| Prefix | Min-Width | Perangkat Umum          |
| ------ | --------- | ----------------------- |
| `sm:`  | 640px     | HP Besar / Tablet Kecil |
| `md:`  | 768px     | Tablet                  |
| `lg:`  | 1024px    | Laptop                  |
| `xl:`  | 1280px    | Desktop                 |
| `2xl:` | 1536px    | Layar Lebar             |

**Contoh:**

```html
<div class="w-full md:w-1/2 lg:w-1/3"></div>
```

## 8. Latar Belakang (Background) & Border

- **Color**: `bg-white`, `bg-red-500`, `bg-transparent`, `bg-current`
- **Gradient**: `bg-gradient-to-r`, `bg-gradient-to-b`, `from-blue-500`, `via-purple-500`, `to-pink-500`

- **Size**: `bg-cover`, `bg-contain`, `bg-auto`
- **Position**: `bg-center`, `bg-top`, `bg-left-bottom`
- **Repeat**: `bg-no-repeat`, `bg-repeat`, `bg-repeat-x`

### Border & Radius

- **Width**: `border`, `border-0`, `border-2`, `border-4`, `border-t-4`
- **Color**: `border-gray-200`, `border-red-500`, `border-transparent`
- **Style**: `border-solid`, `border-dashed`, `border-dotted`, `border-double`, `border-none`
- **Radius**: `rounded-none`, `rounded-sm`, `rounded`, `rounded-md`, `rounded-lg`, `rounded-full` (lingkaran)
- **Divide**: `divide-y`, `divide-x`, `divide-gray-200` (Garis antar elemen child)

### Ring & Outline

- **Ring**: `ring-0`, `ring-2`, `ring-blue-500`, `ring-offset-2` (Glow effect/Focus ring)
- **Outline**: `outline-none`, `outline-dashed`, `outline-offset-2`

## 9. State & Interaktivitas

Mengubah gaya berdasarkan kondisi elemen.

- **Hover**: `hover:bg-blue-700` (Kursor di atas elemen)
- **Focus**: `focus:ring-2`, `focus:outline-none` (Input aktif/dipilih)
- **Active**: `active:scale-95` (Saat diklik/ditekan)
- **Disabled**: `disabled:opacity-50`, `disabled:cursor-not-allowed`
- **Visited**: `visited:text-purple-600` (Link yang sudah dikunjungi)
- **Group**: Berikan class `group` pada parent, lalu `group-hover:text-white` pada child.
- **Peer**: Berikan class `peer` pada elemen sebelumnya, lalu `peer-checked:bg-blue-500` pada elemen sesudahnya (bagus untuk custom checkbox).
- **Dark Mode**: `dark:bg-slate-900`, `dark:text-white` (Aktif saat mode gelap sistem/class menyala).
