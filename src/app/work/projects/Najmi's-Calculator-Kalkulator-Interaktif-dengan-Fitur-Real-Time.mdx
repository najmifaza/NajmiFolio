## Overview

Najmi's Calculator adalah proyek aplikasi berbasis web yang dirancang untuk mengasah kemampuan dasar front-end development. Fokus utama proyek ini adalah penerapan DOM Manipulation dan logika matematika menggunakan JavaScript murni. Selain fungsi hitung, aplikasi ini dilengkapi dengan fitur estetika dan utilitas tambahan seperti sistem waktu real-time dan transisi mode gelap.

- **Operasi Matematika Lengkap**: Mendukung fungsi aritmatika dasar (penjumlahan, pengurangan, perkalian, pembagian) dengan logika yang akurat.
- **Jam & Tanggal Real-Time**: Mengintegrasikan Date() object untuk menampilkan waktu (format 24 jam) dan tanggal dengan lokal id-ID (Indonesia) yang diperbarui secara otomatis setiap detik.
- **Mode Gelap (Dark Mode)**: Fitur toggle dinamis untuk berpindah antara tema terang dan gelap menggunakan transisi CSS yang halus untuk kenyamanan mata pengguna.
- **Custom Error Handling**: Implementasi blok try...catch untuk menangani input matematika yang tidak valid. Alih-alih pesan standar, aplikasi ini menampilkan pesan error yang unik dan personal: "YANG BENER NYED!"
- **Desain Responsif**: Menggunakan CSS Grid dan Flexbox untuk memastikan tata letak tombol tetap presisi dan mudah digunakan di berbagai ukuran layar (mobile & desktop).

## Technologies Used

- **HTML5**: Untuk membangun struktur semantik aplikasi.
- **CSS3**: Untuk desain visual, tata letak grid, dan animasi transisi tema.
- **JavaScript (ES6+)**: Logika utama aplikasi, termasuk penggunaan eval() untuk kalkulasi, setInterval untuk jam, dan manipulasi class untuk fitur Dark Mode.

## Challenges and Learnings

Proyek ini menjadi sarana belajar yang sangat efektif dalam memahami cara kerja asinkronus sederhana (melalui pembaruan waktu detik demi detik) dan penanganan input pengguna. Tantangan terbesarnya adalah memastikan ekspresi matematika yang kompleks tidak merusak alur aplikasi, yang akhirnya berhasil diatasi dengan sistem error handling yang responsif. Saya juga belajar bagaimana memberikan sentuhan personal (humor) ke dalam aspek teknis sebuah produk digital.

## Outcome

Hasilnya adalah sebuah alat bantu hitung yang fungsional sekaligus menarik secara visual. Proyek ini membuktikan pemahaman kuat saya terhadap fundamental JavaScript sebelum melangkah ke framework yang lebih kompleks. Kalkulator ini kini menjadi bagian dari koleksi alat digital pribadi saya yang siap dikembangkan lebih lanjut.
